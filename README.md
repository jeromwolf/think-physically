# ThinkPhysically - 화성행 티켓

> "드라마가 현실이 되는 세상"

## 프로젝트 소개

**화성행 티켓 (Mars Ticket)** - K-SF 드라마/웹툰 프로젝트

2040년, 인류 최초 화성 이주가 시작된다. 선발된 100인, 각자의 이유로 지구를 떠난다. 편도 티켓, 돌아올 수 없는 여정.

**"당신은 왜 화성에 가려 하나요?"**

## 비전

**"세상의 문제를 물리학적으로 해결한다"**

- K-콘텐츠 + 실제 기술 + 창업 영감의 융합
- 드라마가 사업 아이디어의 쇼케이스
- 일론 머스크처럼 생각하기

## 생태계 구조

```
"화성행 티켓" (웹툰/드라마)
        │
        ↓ 기술이 궁금해짐
ThinkPhysically (기술 심화)
        │
        ↓ "나도 만들고 싶다"
커뮤니티 (팀 빌딩)
        │
        ↓ 실제 실행
스타트업 탄생
        │
        ↓
드라마가 현실이 됨
```

## 기술 스택

- **Frontend**: Next.js 14 + TypeScript + Tailwind CSS
- **AI Tools**: Claude, Gemini, ChatGPT, Midjourney, Suno

## 프로젝트 구조
